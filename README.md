# z-uni-template
A wechat mini-program template based on uni-app.


## 一个基于uni-app的微信小程序开发模板
 * 基于vue + ts
 * 基础的base、login等mixin
 * 轻型、双向绑定特性的数据中心
 * 网络请求模块
 * 基础登陆逻辑

### Project setup
```
npm i
```

### Compiles and hot-reloads for development
```
npm start
// 微信开发者工具导入dist/dev/mp-weixin
```

### Compiles and minifies for production
```
npm run build
```
